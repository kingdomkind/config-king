# Official Table

### Example Definitions:
```lua
Packages = {
    Official = {

    }
}
```

```lua
Packages = {
    Official = {
        "base",
        "base-devel",
        "git",
    }
}
```

### Element Types:

**Regular Package:**
```lua
"package-name-here",
```
Simply put the package name, and it will install the package from official arch repos.
