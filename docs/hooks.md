# Build Hooks

### Example Definition:
```lua
function HookPost()
    print("Lemon cheese")
end
```

**Possible Hooks:**
- HookPost
    - Arguments Passed: None
    - Ran once the rest of the build script is completed

*Please note that lua code not in a function is arbitrarily executed at the start of the build script*
